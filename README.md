# GameTracker Frontend

Aplicación frontend para gestionar tu biblioteca personal de videojuegos.

## Características
- Ver biblioteca de juegos
- Agregar y editar juegos
- Escribir reseñas
- Registrar horas jugadas
- Puntuación con estrellas

## Tecnologías
- React
- CSS puro
- API REST

## Instalación
1. npm install
2. npm start
